# Wedding Invitation

An elegant online wedding invitation website.

## Features
- Beautiful, responsive design
- Guest RSVP functionality
- Event details and location
- Photo gallery
- Gift registry links

## Technologies
- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize with your wedding details

## Customization

Edit `index.html` and `style.css` to add:
- Couple names
- Wedding date and time
- Venue details
- Wedding party info
- Photos and colors

## Deployment

Deploy to GitHub Pages or any web hosting service.
